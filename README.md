# Getting Started on Local
To run this project on your local machine:

- Clone this repository.
  ```bash
  git clone https://github.com/Ifcodes/star-wars-xters.git
  ```

- Install all dependencies using 

```bash 
  yarn install
```
- Run

```bash
yarn dev
```
